# Code Quality Dashboard

## Features
- Analyzes code quality using Claude AI
- Provides scores for architecture, readability, performance
- Detects design patterns and suggests refactoring
- Rate-limited to prevent abuse

## Architecture
- **Frontend:** Next.js + React + TypeScript
- **Backend:** Node.js serverless functions
- **AI:** Claude/OpenAI API
- **Deployment:** Netlify


## Running Locally
```bash
npm install
npm run dev
```
